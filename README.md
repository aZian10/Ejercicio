# Fusion Gourmet Landing Page

Landing page built with HTML5 and CSS3, reorganized with a modular architecture.

## Project Architecture

```text
public/
├── img/
├── audio/
├── videos/
├── icons/
└── docs/

src/
├── css/
│   ├── base/
│   ├── layout/
│   ├── components/
│   ├── pages/
│   └── style.css
├── views/
│   └── mascotas.html

index.html
README.md
LICENSE.md
```

## Technologies
- HTML5
- CSS3

## How to Run
1. Open `index.html` in your browser.
2. For YouTube embeds, prefer running it with a local server instead of `file://`.
